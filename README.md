- **Image Cards**: White cards with hover effects and metadata

### Responsive Breakpoints
- **Desktop**: 1200px+ (4 columns)
- **Tablet**: 768px-1199px (2-3 columns)
- **Mobile**: 480px-767px (1-2 columns)
- **Small Mobile**: <480px (1 column)
